Age 3+ 15 min No mess

Tape Spider Web

Five minutes to build. Twenty runs to get through without touching a strand.

What you need

Colored tape. Two or more chairs.

Steps
1

Place two chairs facing each other about three feet apart. Pull them close enough that tape can stretch between the legs.

2

Run tape strands back and forth between the chair legs at different angles and heights until you have a web. Crisscross, diagonal, whatever fills the space.

3

The rule: get from one side to the other without touching any tape.

4

Touch a strand and you restart, or take a five-second penalty if you want to keep moving.

Variations
Timer run

No elimination, just clock how fast they can get through clean.

Builder runs it

Whoever built the web has to run it. Make it hard on yourself.

Bigger web

Add more chairs to expand the course into a full room obstacle.

Parent tip: Build it harder than you think they can handle. The run they finally complete clean after six attempts is the one they will talk about at dinner.
